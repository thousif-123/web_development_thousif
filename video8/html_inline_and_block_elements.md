# Commonly Used Inline and Block Elements in HTML

This document lists the most commonly used **block-level** and **inline** HTML elements along with their typical use cases.

---

## Block-Level Elements

Block elements usually start on a new line and take up the full width available.

### `<div>`
**Use case:** Generic container for grouping elements, layout structure.

### `<p>`
**Use case:** Paragraph of text.

### `<h1>` â€“ `<h6>`
**Use case:** Headings (h1 is most important, h6 least).

### `<section>`
**Use case:** Thematic grouping of related content.

### `<article>`
**Use case:** Independent, self-contained content (blog post, news article).

### `<header>`
**Use case:** Introductory content or navigation for a section or page.

### `<footer>`
**Use case:** Footer information (copyright, links).

### `<nav>`
**Use case:** Navigation links.

### `<ul>`
**Use case:** Unordered (bulleted) list.

### `<ol>`
**Use case:** Ordered (numbered) list.

### `<li>`
**Use case:** List item (used inside ul/ol).

### `<main>`
**Use case:** Main content of the page (unique).

### `<form>`
**Use case:** Container for user input fields.

### `<table>`
**Use case:** Display tabular data.

### `<blockquote>`
**Use case:** Large quoted section.

---

## Inline Elements

Inline elements do not start on a new line and only take as much width as needed.

### `<span>`
**Use case:** Generic inline container for styling text.

### `<a>`
**Use case:** Hyperlink to another page or section.

### `<img>`
**Use case:** Display images.

### `<strong>`
**Use case:** Important text (semantic emphasis).

### `<em>`
**Use case:** Emphasized text (usually italic).

### `<b>`
**Use case:** Bold text (visual only).

### `<i>`
**Use case:** Italic text (visual only).

### `<u>`
**Use case:** Underlined text.

### `<small>`
**Use case:** Small print or side comments.

### `<code>`
**Use case:** Inline code snippet.

### `<label>`
**Use case:** Label for form inputs.

### `<input>`
**Use case:** User input field (text, checkbox, etc.).

### `<button>`
**Use case:** Clickable button.

### `<br>`
**Use case:** Line break.

---

## Key Difference Summary

| Block Elements | Inline Elements |
|---------------|----------------|
| Start on a new line | Stay in the same line |
| Take full width | Take content width |
| Can contain inline & block | Usually contain text or inline |

---

## Tip for Beginners

- Use **block elements** for structure and layout.
- Use **inline elements** for styling or small interactions inside text.
